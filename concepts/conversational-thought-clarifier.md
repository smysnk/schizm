# Conversational Thought Clarifier

## Core Idea

Add a voice or Alexa-style interface for capturing thoughts through conversation instead of typing alone.

## Why It Matters

Follow-up questions can improve two things at once:

- the system gets better metadata for organizing the idea
- the person gets help thinking the idea through

## Interaction Pattern

1. A thought is captured in a short, natural way.
2. The system asks a small number of follow-up questions.
3. The answers clarify meaning, intent, and context.
4. The final note is organized with stronger structure than the original input alone.

## Design Constraints

- Keep the conversation lightweight so capture still feels easy.
- Ask only questions that improve recall, structure, or conceptual clarity.
- Treat the dialogue as part of the note history, not just hidden system processing.

## Relationship To The Inbox

This extends the thought-capture inbox rather than replacing it.

Typing supports fast dumping. Conversation supports clarification. Both feed the same knowledge system.
