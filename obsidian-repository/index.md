## Current Focus

This repository starts with an idea about making thought capture frictionless: a simple place to drop unfinished thoughts and let the system organize them later.

The follow-on concept is a conversational interface that does not just store ideas, but asks clarifying questions so the thought becomes easier to understand, sort, and revisit.
