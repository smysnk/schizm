# Thought Capture Inbox

## Core Idea

Create a single prompt box where random thoughts can be captured before they disappear.

The interface should optimize for low friction:

- one obvious place to type
- no required categorization up front
- the system organizes notes after capture instead of before capture

## Desired Behavior

- Accept messy, partial, and unstructured thoughts.
- Group related ideas over time.
- Make earlier thoughts discoverable instead of buried in a raw log.
- Preserve the original wording alongside any later structure.

## Product Shape

This behaves like an inbox for thinking rather than a traditional notes app.

The main value is not just storage. It is reducing the mental cost of capture while building a structure that becomes more useful later.
